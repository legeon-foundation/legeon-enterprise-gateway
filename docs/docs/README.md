# Legeon Enterprise Gateway — Documentation Index

This directory contains the technical documentation for the Catalyst-funded
S2F (Source → Verify → Fulfill) module of the Legeon Enterprise Gateway.

## Included in Catalyst Scope
- Requisition intake  
- Shortlist retrieval (via Legeon Core)  
- Candidate verification (via CompactDApp proofs)  
- Award workflow  
- Statement of Work (SoW) generation and approval  
- Optional Award sync to SAP
